# Background Music Setup

To add background music to your videos, download royalty-free tracks and place them in this folder.

## Recommended Sources:

1. **YouTube Audio Library**
   - URL: https://studio.youtube.com/channel/UC_YOUR_CHANNEL/music
   - Genre: Ambient, Cinematic, Electronic
   - Format: MP3

2. **Pixabay Music**
   - URL: https://pixabay.com/music/
   - Free for commercial use
   - No attribution required

3. **Free Music Archive**
   - URL: https://freemusicarchive.org/
   - Filter by Creative Commons licenses

## File Format:
- **Supported**: MP3, WAV, M4A
- **Recommended**: MP3 for smaller file size
- **Volume**: Will be auto-adjusted to -20dB

## Usage:
The system will randomly select a track from this folder for each video. The music will be:
- Mixed with the voiceover
- Ducked to -20dB to not overpower the voice
- Looped if shorter than the video duration
